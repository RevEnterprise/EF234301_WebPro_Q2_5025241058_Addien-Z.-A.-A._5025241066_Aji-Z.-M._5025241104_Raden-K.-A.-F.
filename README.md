### Dependency for Local Deployment
```
JDK 21.0.8
Maven 3.9.9
Tomcat10
```

### Local deployment
1. Git clone this repo
2. Go to the new directory
3. Build the project using maven `mvn clean package`
4. Copy the warfile in `target/` directory to tomcat `webapps/` directory
5. Open the website in localhost
